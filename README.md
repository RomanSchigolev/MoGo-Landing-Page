# MoGo

MoGo - учебный макет для верстки. Этот макет включает в себя такие компоненты, как: 
* аккордеон, 
* карусель с отзывами, 
* модальное окно, 
* мобильное меню. 

Были использованы flexBox и Grid.
Реализована адаптивность под мобильные устройства.
